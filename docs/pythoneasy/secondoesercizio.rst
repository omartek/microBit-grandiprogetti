Esercizi con i numeri
=====================

Primo esercizio: le moltiplicazioni
-----------------------------------

Scriviamo un semplice moltiplicazione::

  print (3*5);
  print (4*5);

E una serie di moltiplicazioni::

  # tabelline
  for a in range(0,11):
    for b in range(0,11):
      print (a,"*",b,"=",a*b)
