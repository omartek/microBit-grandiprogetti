Puoi utilizzare il linguaggio Python scaricando il software necessario IDLE e l'interprete dal sito ufficiale, eventualmente c'è anche la versione italiana:

* `download Python [en] <https://www.python.org/downloads/>`_
* `download Python [it] <http://python.it/download/>`_
oppure usarlo online e, volendo, seguendo i tutorial proposti qui:

* `Trinket.io <https://trinket.io/>`_

Esiste anche uno strumento a blocchi per addestrarsi con Python, lo trovi qui:

* `Edublocks <https://edublocks.org/#raspberrypi>`_

Consulta il tutorial contenuto nel sito ufficiale o i testi consigliati nella versione italiana:

* `tutorial ufficiale [en] <https://docs.python.org/3.5/index.html>`_
* `testi da consultare [it] <http://python.it/doc/libri/>`_

Consiglio questi due:

* `Pensare come un informatico <https://github.com/AllenDowney/ThinkPythonItalian/blob/master/thinkpython_italian.pdf>`_
* :download:`Python passo passo <./manuale_python_passopasso/manuale-python-V2.pdf>`.
