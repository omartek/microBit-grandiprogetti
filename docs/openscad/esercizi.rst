Costruisci un robot umanoide
****************************

Costruisci un robot umanoide usando ``cube (cubo)`` e ``cylinder (cilindro)``, traslandoli oportunamente.
Il robot umanoide deve avere:
* due gambe
* un busto
* due braccia
* un collo
* una testa

::

  //gambe
    cylinder([]);
